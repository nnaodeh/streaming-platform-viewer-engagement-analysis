# Streaming Platform Viewer Engagement Analysis

## Project Overview

This project analyses viewer behaviour on a streaming platform to identify the content genres that drive the highest engagement, repeat viewership, and subscriber retention.

Using Microsoft Excel, I transformed raw user, subscription, and viewing activity data into an interactive dashboard that provides insights to support content investment and marketing decisions.

## Business Problem

StreamWave Entertainment needed to understand which content genres perform best in order to:

* Improve viewer engagement
* Increase subscriber retention
* Reduce churn
* Optimise content investment
* Support data-driven decision-making

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

## Process

1. Cleaned and prepared the data.
2. Analysed viewing activity across genres.
3. Measured engagement and repeat viewership.
4. Examined genre impact on subscriber retention.
5. Built an interactive dashboard for reporting and decision-making.
