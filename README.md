\# Streaming Platform Viewer Engagement Analysis Dashboard



\## Project Overview



This project analyses viewer engagement patterns on a streaming platform using Microsoft Excel.



The dashboard was developed to identify viewing trends, user engagement behaviour, content performance, and retention patterns.



\## Tools Used



\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Slicers

\- Data Cleaning

\- Dashboard Design



\## Dataset



The project analyses:



\- User data

\- Subscription data

\- Viewing activity records

\- Movie catalogue information



\## Key Insights



\- Viewer engagement trends

\- Genre performance analysis

\- Content completion rates

\- Repeat viewing behaviour

\- Retention metrics
